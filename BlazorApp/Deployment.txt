
CD path from Github Release branch to Azure website juoksulaskuri.azurewebsites.net

Deployment
- Finish changes and commit to main
- Change to release branch (some errors might popup to VisualStudio -> ignore)
- Branch - Update from main (errors should go away)
- Commit to Release -> CD will automatically publish website
- Open history and create a Tag for the release
- Change back to Main branch
